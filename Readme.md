## Tech Stack

**Server:** Golang <br>
**Database:** Postgres

## Inisiasi

#### Inisiasi Package

```
go get github.com/gin-gonic/gin
go get golang.org/x/crypto
go get gorm.io/driver/postgres
go get gorm.io/gorm
go get github.com/golang-jwt/jwt
```

#### development

```
go run main.go
```

#### build production

```
go build
```
